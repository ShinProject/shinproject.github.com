1 Persyaratan Sistem

Sebelum menginstal, pastikan:

  • RAM: Minimal 2GB (rekomendasi 4GB+ untuk 50+ mod)
  • Java: Versi 8 atau terbaru (64-bit)
  • Minecraft: Versi 1.12.2 sudah terinstal
  • Forge: Versi 14.23.5.2860 atau lebih baru

2 Download Modpack

Klik tombol "Download Modpack" di atas untuk mendapatkan file ZIP berisi:

  • Folder /mods (berisi semua mod)
  • Folder /config (konfigurasi optimal)
  • Folder /scripts (opsional)

3 Instalasi Forge

Jika belum memiliki Forge:

  1. Download installer dari situs resmi Forge
  2. Jalankan file .jar yang didownload
  3. Pilih "Install client" dan klik OK
  4. Tunggu hingga proses selesai

4 Ekstrak Modpack

Salin semua isi folder dari file ZIP ke direktori Minecraft:

android/data/minecraft/.minecraft

Cara cepat (pc):

  1. Tekan Win + R
  2. Ketik %appdata%\.minecraft
  3. Tekan Enter
  4. Salin semua file dari folder modpack

5 Memulai Permainan

Setelah semua file ditempatkan:

  1. Buka Minecraft Launcher
  2. Pilih profil "Forge"
  3. Pastikan versi 1.12.2
  4. Klik "Play"
  5. Tunggu hingga game selesai memuat semua mod
